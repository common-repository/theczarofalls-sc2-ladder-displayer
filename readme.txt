== Description ==

Displays StarCraft 2 ladder rank and other vital information as a widget.

== Installation ==

Place all files and folders in the Plugin directory. Go to your Appearance>Widgets section of the Admin area and set your Battle.net profile URL and your region. 
Note: for NA accounts, use en/us. For Kr/Tw accounts use the KR site. For all eu accounts, use the English site.
